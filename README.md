# 2025 서원대학교 미디어콘텐츠학부 졸업전시회 홈페이지

본 프로젝트는 2025 졸업전시회 슬로건 “RE: NEW MYSELF — A STORY OF OUR LIFE : REWRITE”를 중심으로  
전시 정보·학생 참여·작품을 웹에서 몰입감 있게 관람할 수 있도록 제작된 공식 홈페이지입니다.

사이트: **https://media.seowon.ac.kr/2025**

---

## 주요 기능
- **React Router 기반 전시 페이지 라우팅**
- **웹, 모바일 반응형 구조**
- **메인 풀스크린 영상 + 스크롤 인식 인터랙션**
- **SVG 라인 드로잉 기반 컨셉 섹션**
- **공식 포스터 안내 섹션**
- **3D 멤버 캐러셀 (PC) / 자동 슬라이더 (모바일)**
- **IntersectionObserver 기반 스크롤 등장 애니메이션**
- **챗봇 + Top 버튼 플로팅 UI**

---

## 기술 스택
- **Frontend:** React, JavaScript, CSS
- **Interaction:** IntersectionObserver, SVG Path Animation, 3D Transform
- **Deploy:** React 정적 빌드 결과물을 학교 웹 서버에 FileZilla로 업로드하여 제공 (정적 호스팅)
